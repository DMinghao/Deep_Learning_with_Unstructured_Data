## Getting Started

### Using Virtual Environment

- create virtual env

    ```bash
    $ python -m venv venv
    ```
- run virtual env
    - Windows
    ```bash
    $ venv\Scripts\activate.bat
    ```
    - Mac / Linux
    ```bash
    $ source venv/bin/activate
    ```
- exit virtual env
    ```bash 
    $ deactivate 
    ```

### Requirements 

- General Requirements 
    ```
    Java        == 1.8 
    Gephi       >= 0.92
    ```
